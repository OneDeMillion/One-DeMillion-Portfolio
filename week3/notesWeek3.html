Map/Filter/Reduce

  • Map() method creates a new array from an existing one, it applies a function to each one of the elements in the original array
  • const intNum = [1,2,3,4);
    const doubleNum = intNum.map(item => item * 2);
    console.log(doubleNum); // [2,4,6,8]
  • Filter() method takes each element of the original array and applies a conditional statement. If true, the element is pushed to the new array.
  • const intNum = [1,2,3,4);
    const evenNum = intNum.filter(item => item % 2 === 0);
    console.log(evenNum); // [2,4]
  • Reduce() method reduced the array to just one single value. To get that value it runs a function on each element of the original function.
  • const trees = [oak, fir, maple, maple, aspen, oak, fir, oak, maple, fir);
    const treeCount = trees.reduce(function(obj, tree) {
    if (!obj[tree]) {
    obj[tree] = 1;
    } else {
    obj[tree]++;
    }
    return obj;
    }, {});

    console.log(treeCount);

    Output:
    {
    oak: 3; fir: 3; maple: 3; aspen: 1;
    }
