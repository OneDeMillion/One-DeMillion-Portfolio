Notes on Reading

Color Pickers

  • If you want to show a color picker in your code, use:
    type="color" in input fields that should contain color

Element Method Query Selector All

  • Returns a static Node List
  • Looks like an Array but it isn't
  • Nodelists only support a few methods, such as:
    item(), entries(), keys(), values(), forEach
  • If you want to filter your nodelist, your're in trouble, unless you convert it to an array.
  • Easy way to do this is with Array.from()
          const AllTheDogsArray = Array.from(AllTheDogs);
